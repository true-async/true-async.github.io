## 简介 {#introduction}

`PHP TrueAsync` 是一个通过修改 Zend 内核、I/O 库、数据库库、套接字库以及其他函数来实现 PHP 真正异步的项目。

`PHP TrueAsync` 实现了无着色函数的透明异步范式，
最大限度地减少了代码更改，并消除了库的碎片化。
换句话说，使用协程时，您使用相同的函数，无需更改或仅需最少的更改。
