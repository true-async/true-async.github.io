## Вступ {#introduction}

`PHP TrueAsync` — проект, що реалізує справжню асинхронність у PHP шляхом зміни ядра Zend, бібліотеки введення-виведення,
бібліотек для роботи з базами даних, сокетами, а також інших функцій.

`PHP TrueAsync` реалізує парадигму прозорої асинхронності без кольорових функцій,
яка мінімізує зміни в коді та усуває сегментованість бібліотек.
Іншими словами, використовуючи корутини, ви використовуєте ті самі функції без змін або з мінімальними змінами.
