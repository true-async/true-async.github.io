## 소개 {#introduction}

`PHP TrueAsync`는 Zend 코어, I/O 라이브러리,
데이터베이스 라이브러리, 소켓 라이브러리 및 기타 함수를 수정하여 PHP에서 진정한 비동기를 구현하는 프로젝트입니다.

`PHP TrueAsync`는 컬러드 함수 없이 투명한 비동기 패러다임을 구현하여
코드 변경을 최소화하고 라이브러리 분절을 제거합니다.
즉, 코루틴을 사용할 때 변경 없이 또는 최소한의 변경으로 동일한 함수를 사용합니다.

## 설치 {#installation}

`PHP TrueAsync`는 세 가지 방식으로 사용할 수 있습니다:
* Docker 이미지
* 인기 배포판용 사전 빌드 패키지
* 수동 빌드를 위한 소스 코드
